Brackets Pair Colorizer

Famosa extensão que é bastante útil para diferenciar com cores diferentes os pares de colchetes no código, tornando melhor a visualização dos vários colchetes encadeados que podem surgir.

GitLens

Esta extensão ajudará no trabalho em equipe, já que ela promove a visualização de alterações e adições feitas no código, trazendo os diffs de cada código e uma sessão exclusiva do GitLens, que agrega as informações mais importantes do repositório, como os commits feitos, quais branches, tags, etc.

Vscode icons

Esta extensão cria ícones referentes a cada tecnologia que for usada no projeto, ajudando na organização e na melhor identificação do mesmo.

Auto import

Nesta extensão, os imports serão localizados, analisados e irão autocompletar a importação que está sendo chamada.

Beautify

É um dos plugins mais famosos dos desenvolvedores, ele é um embelezador de código, tornando ele identado para arquivosjson, html, css, Sass e javascript.

Live Server

Uma das extensões obrigatórias para quem trabalha com front-end através do Visual Studio Code.

Porque devo aprender HTML?

Criar sites: HTML é uma habilidade fundamental para quem quer criar sites. Se você está interessado em desenvolver sites, aprender HTML é essencial. Saber como escrever código HTML significa que você pode construir sites desde o início, o que é uma habilidade valiosa para ter.

Personalização: Quando você sabe HTML, pode personalizar seus próprios sites. Você pode criar layouts personalizados, escolher as cores e fontes que deseja usar e adicionar recursos que não estão disponíveis nos editores de site padrão. Isso significa que você pode criar um site exclusivo e exclusivo que se destaca na multidão.

Melhorar o SEO: O HTML também pode ajudar a melhorar o SEO do seu site. SEO significa "Otimização de Mecanismos de Busca", que é a prática de otimizar seu site para que ele apareça mais alto nos resultados dos mecanismos de busca como o Google. Ao escrever código HTML limpo e semântico, você pode ajudar os motores de busca a entender do que se trata seu site e como ele deve ser classificado nos resultados de pesquisa.
